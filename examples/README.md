# Programas de ejemplo — Lenguaje de Control de Accesos

Estos archivos `.acl` son programas completos que demuestran el uso de las categorías de tokens y la estructura del lenguaje.

| Archivo | Contenido |
|---------|-----------|
| `01_roles_y_usuarios.acl` | Definir Rol/Usuario/Modulo; asignación de roles y usuarios. |
| `02_reglas_seguridad.acl` | Permitir, Denegar, Acceder, Validar; operaciones Consultar, Registrar, etc. |
| `03_control_flujo.acl` | Si/Entonces/Sino, Mientras, Elegir/Caso/Terminar. |
| `04_manejo_errores_y_salida.acl` | Intentar/Atrapar/Error; Mostrar y Devolver. |
| `05_programa_completo.acl` | Ejemplo que combina definiciones, reglas, condicionales, bucles y manejo de errores. |

Para probarlos en el analizador léxico: **Cargar archivo** (📂) y elegir uno de estos `.acl`, luego **▶ Analizar**.
